Tally Matrix Clock

Matrix-style time display for Apple TV

Live on App Store tvOS
Download on App Store View on GitHub

About

Tally Matrix Clock transforms your Apple TV into a unique visual timepiece. Time is displayed through illuminated colored squares in a matrix format, where the number of lit squares equals each digit's value.

Each digit reorganizes randomly at your chosen interval, creating an ever-changing display that combines retro matrix aesthetics with modern design.

Key Features

Matrix Display

Four matrices represent time digits with illuminated squares

Multiple Formats

12-hour or 24-hour time display

Color Schemes

Random per square, per matrix, or single color options

Dynamic Patterns

Squares reorganize at 5s, 15s, 30s, or 60s intervals

Privacy First

Zero data collection, all settings stored locally

Apple TV Optimized

Designed specifically for the big screen
