The NightGard System: Self-Hosted Web Services

November 2025 Projects Michael Lee Fluharty

NightGard is my answer to the complexity of modern web hosting. It's a suite of macOS tools designed to make self-hosting simple and accessible. The core wWw NGPortal application recently achieved MVP status‚Äîa fully functional local web server with DDNS integration.

The goal is to give individual developers and hobbyists the power to host their own content without relying on third-party services. Privacy, control, and simplicity are the driving principles.

Why Self-Hosting Matters

We've become too dependent on third-party hosting services. GitHub Pages, Netlify, Vercel‚Äîthey're all great, but they come with limitations, tracking, and the risk of platform changes.

Self-hosting puts you back in control:

  • Your data stays on your hardware
  • No platform lock-in or service changes
  • Complete control over your infrastructure
  • Zero recurring costs beyond your internet connection

The wWw NGPortal

The first component of NightGard is wWw NGPortal‚Äîa macOS application that makes running a local web server as simple as clicking a button.

Key Features:

  • Static File Serving - Drop your HTML/CSS/JS files in a folder and serve them instantly
  • DDNS Integration - Built-in DuckDNS support to give your home server a domain name
  • Zero Configuration - Works out of the box with sensible defaults
  • Un-sandboxed - Full system access for maximum flexibility

Technical Foundation

NGPortal is built on Swift and the Vapor web framework. This gives us:

  • Native macOS performance
  • Modern Swift concurrency
  • Battle-tested HTTP server
  • Easy extensibility for future features

MVP Achieved: November 2025

After months of development and iteration, wWw NGPortal reached MVP status on November 1, 2025. The core features work reliably:

  • ‚úÖ Web server starts and serves files correctly
  • ‚úÖ Accessible across local network (0.0.0.0 binding)
  • ‚úÖ DDNS updater framework in place
  • ‚úÖ Clean UI with status indicators
  • ‚úÖ Reliable startup and shutdown

What's Next

The NightGard ecosystem is just beginning. Future components include:

  • NGWebPortal - CMS-like tool for blogs and portfolios
  • NightGard BBS - Modern take on bulletin board systems
  • Enhanced DDNS - Support for more providers beyond DuckDNS
  • Backup Tools - Automated backup and sync utilities

Philosophy: Simple, Not Simplistic

NightGard aims to be simple to use, but not simplistic in capability. The goal is to hide complexity, not remove features.

A beginner should be able to get a web server running in minutes. An advanced user should have access to configuration and customization.
